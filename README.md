# MCC Manager

Professional platform for managing Google Ads MCC accounts.

## Features

- MCC account management
- Google Ads account creation
- Metrics tracking (clicks, impressions, cost, conversions)
- Multi-MCC support
- Account transfers between MCCs

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (Database + Edge Functions)
- Google Ads API

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deploy the `dist` folder to your hosting provider.
